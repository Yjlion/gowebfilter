Proxy Authentication

Require clients to present a username and password before they can use the proxy (HTTP Basic, RFC 7235). Applies to HTTP proxy requests and HTTPS CONNECT tunnels.

Password is stored salted + hashed (PBKDF2). Credentials travel over the network as Base64 (HTTP Basic) — use on a trusted LAN or behind HTTPS/VPN.

Upstream Proxy

When set, all regular-mode listeners forward through this parent proxy. Restart the proxy after changing.

TUN / tun2socks

Capture this machine's traffic through a TUN interface and send it into the local SOCKS5 proxy. Needs administrator/root privileges. Restart after saving changes.
