DNS over HTTPS (DOH)
Block domains via a DOH resolver (e.g. family/malware filters)

Pick a preset or type any standard DoH endpoint (RFC 8484), e.g. NextDNS https://dns.nextdns.io/<your-id>. A domain is blocked when the resolver returns NXDOMAIN or sinkholes it (0.0.0.0 / ::).

Note: DoH queries leave the proxy as outbound HTTPS to the resolver. A restrictive network firewall may block or intercept them — if DoH filtering seems inactive, confirm the proxy host can reach the resolver on port 443.
